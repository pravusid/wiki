        # XML DTD에 정의된 태그들

        • 최상위 태그(a, b, c, d, e) : 사용 순서
        • a? : 사용강제X
        • b* : 사용강제X & 여러번 사용가능
        • c+ : 반드시 한 번 이상 사용
        • d : 반드시 한 번만 사용
        • e|k : 둘 중 하나는 반드시 사용
