        # supervisor

        http://supervisord.org/index.html (opens new window)

        python으로 제작된 프로세스 모니터링/관리 도구이다

        # 설치

        sudo apt-get install supervisor

        # 설정

        /etc/supervisor/conf.d/<프로그램명>.conf

        [program:&lt;프로그램명>]
environment=&lt;변수1>=&lt;내용1>,&lt;변수2>=&lt;내용2>
command=/bin/cat
process_name=%(program_name)s // process_name expr (default %(program_name)s)
numprocs=1 // number of processes copies to start (def 1)
user=ubuntu

        http://supervisord.org/running.html?highlight=reread#supervisorctl-actions (opens new window)

sudo supervisorctl start all
sudo supervisorctl status
