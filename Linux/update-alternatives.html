      # update-alternatives

      여러 버전/종류의 패키지를 설치했을때 동적으로 변경하기위해 사용한다

      sudo update-alternatives --config <대상>

      • sudo update-alternatives --config java

      # 터미널 에뮬레이터 변경

      sudo update-alternatives --config x-terminal-emulator
